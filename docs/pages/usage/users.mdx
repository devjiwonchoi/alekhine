# Users

> Lichess API Docs - [Users](https://lichess.org/api#tag/Users)

Equine supports the `Users` API as `users` and `user`.

## `users`

### Get users by IDs

```ts
client.users.info({ ids: string | string[] })
```

### Get real-time users status

```ts
client.users.status({ ids: string | string[] })
```

### Get crosstable

```ts
client.users.crosstable({ user1: string, user2: string, matchup: boolean })
```

### Get live streamers

```ts
client.users.streaming()
```

## `user`

### Get user public data

```ts
client.user.info(username: string)
```

### Get rating history of a user

```ts
client.user.history(username: string)
```

### Get performance statistics of a user

```ts
client.user.performance({ username: string, perfType: string })
```

### Get user activity

```ts
client.user.activity(username: string)
```

### Autocomplete usernames

```ts
// TODO: specify types
client.user.autocomplete({ term, details, friendPrior })
```

### Add a note for a user

```ts
client.user.note({ username: string, text: string })
```

### Get notes for a user

```ts
client.user.notes({ username: string })
```

### Follow a player

> This feature is under the [Relations](https://lichess.org/api#tag/Relations/operation/followUser) part.

```ts
client.user.follow(username: string)
```

### Unfollow a player

> This feature is under the [Relations](https://lichess.org/api#tag/Relations/operation/unfollowUser) part.

```ts
client.user.unfollow(username: string)
```
