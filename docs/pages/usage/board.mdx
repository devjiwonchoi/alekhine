# Board

> Lichess API Docs - [Board](https://lichess.org/api#tag/Board)

### Stream incoming events

```ts
client.board.events()
```

### Stream Board game state

```ts
client.board.stream(gameId: string)
```

### Make a Board move

```ts
client.board.move({ gameId: string, move: string })
```

### Write in the chat

```ts
client.board.chat({
  gameId: string,
  room: 'player' | 'spectator',
  text: string,
})
```

### Fetch the game chat

```ts
client.board.chat.fetch({ gameId: string })
```

### Abort a game

```ts
client.board.abort(gameId: string)
```

### Resign a game

```ts
client.board.resign(gameId: string)
```

### Handle draw offers

```ts
client.board.draw({ gameId: string, accept: boolean })
```

### Handle takeback offers

```ts
client.board.takeback({ gameId: string, accept: boolean })
```

### Claim victory of a game

```ts
client.board.victory(gameId: string)
```

### Berserk a tournament game

```ts
client.board.BERSERK(gameId: string)
```
