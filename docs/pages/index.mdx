# What is Equine?

Equine is minimal, zero-config, asynchronous [Lichess API](https://lichess.org/api) Client written in TypeScript.

## Basic Usage

```bash
pnpm i equine
```

### CommonJS (CJS)
  
```js
const { Client } = require('equine')

const client = new Client()

await client.//...
```

### ES Modules (ESM)

```js
import { Client } from 'equine'

const client = new Client()

await client.//...
```
